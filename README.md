# Bluetooth & Automatic Controlled Programmable Firefighter Robot

## Overview
A smart firefighting robot capable of detecting and extinguishing fire automatically using flame sensors and a water pump. The robot can also be controlled manually through a Bluetooth mobile application.

## Features
- Automatic fire detection
- Automatic fire suppression
- Bluetooth-based manual control
- Dual operating modes

## Technologies Used
- Arduino Uno
- C++ Programming
- HC-05 Bluetooth Module
- Flame Sensors
- Relay Module
- Water Pump

## Skills
- Arduino Programming
- C++ Programming
- Embedded Systems
- Bluetooth Communication
- Circuit Design & Soldering
